  • [컨텐츠] 가게등록은 어떤 과정을 거쳐 공개되나요?

    Eat it은 '사전 필터링' 과정을 거쳐 게시물을 공개하고 있습니다.
    이 과정은 게시물의 "사실여부"를 사전에 확인하거나 "옳고 그름의 가치판단"을 하는 과정이 아닙니다.
    저희팀은 게시물을 관리하는 고유한 운영정책을 보유하고 있으며, 모든 필터링 과정은 전문 부서에서 일임하여 진행하고 있습니다.


    게시물 수정을 요청드리는 가장 대표적인 사례는 아래와 같습니다.
    - 다수의 회원들이 함께 이용하는 공간인만큼, 욕설/비속어와 같이 '금칙어'로 지정한 단어가 포함된 게시물
    - 분량 채우기를 목적으로, 정보성 컨텐츠가 아니라 노래가사, 뉴스기사, 게시물 작성 가이드 예시문을 작성한 게시물
    - 컨텐츠 종류가 바뀌어 작성된 경우,
    이외에도, 운영정책 상 그대로 공개가 어려운 게시물은 수정을 요청드릴 수 있습니다.

  • [컨텐츠] 내가 작성한 글을 어떻게 수정/삭제하나요?

    - 먼저 로그인 후 ~ 확인하시고 ~를 선택하시기 바랍니다.
    그래도 안될 경우 문의하기를 통해 연락주시면 안내도와드리겠습니다.

  • [계정] 업체등록은 어떻게 하나요?

    - 먼저 로그인 후 왼쪽 최상단의 三을 선택 후, 가게등록신청으로 이동하여 작성 후 등록바랍니다.
    그래도 안될 경우 문의하기를 통해 연락주시면 안내도와드리겠습니다.

  • [신고] 승인이 거절되었다는 것은 무엇인가요?"

    - 검토 결과, 운영정책에 맞지 않아 승인이 거절되었습니다.

  • [계정] 회원탈퇴는 어떻게 하나요?

    - 회원탈퇴는 사이트 내 '계정' 메뉴에서 '회원탈퇴'를 클릭하여 요청하실 수 있습니다.

    - FRONT : HTML5, JQUERY
       BACK : JAVA
       개발환경 : Oracle JDK 11, ORACLE 11, Tomcat 9.0, Spring boot 4,

    - 노션을 통해 일정을 계획하고 진행했지만,

    - 프론트엔드와 백엔드 프로그램 스킬이 향상됨을 느꼈습니다.
